# 브베 열샤 앱 개발(Server)

## 대표적인 기능

- 브베 전자도네 모음 영상 재생 기능
- 열샤 전용 단체 톡방 기능
- 브베 방송 도방 기능

## 설치 모듈

### NestJS

```
$ npm i -g @nestjs/cli
$ nest new server
```

## Mongoose

```
$ npm i @nestjs/mongoose mongoose
```

## Config

```
$ npm i -g @nestjs/cli
```

## Class Validator

```
$ npm i --save class-validator class-transformer
```
