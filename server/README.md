# 브베 열샤 앱 개발(Server)

## 대표적인 기능

- 브베 전자도네 모음 영상 재생 기능
- 열샤 전용 단체 톡방 기능
- 브베 방송 도방 기능

## 설치 모듈

### NestJS

```
$ npm install -g @nestjs/cli
$ nest new server
```

## Mongoose

```
$ npm install @nestjs/mongoose mongoose
```

## Config

```
$ npm install -g @nestjs/cli
```

## Class Validator

```
$ npm install --save class-validator class-transformer
```

## Swagger

```
$ npm install --save @nestjs/swagger
```

## Bcrypt
```
$ npm install bcrypt
$ npm install @nestjs/bcrypt
```
